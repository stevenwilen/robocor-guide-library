# Scope Check Skill

Classify each requested feature as one of:

## In current scope
- course directory
- Morpheus Drive course page
- complete Lesson 1
- placeholder Lessons 2 and 3
- reusable course/lesson layout
- local progress
- responsive design
- deployment-ready static frontend

## Future paid add-on
- new course content
- full Lesson 2 or Lesson 3 content for the updated app
- new app screenshots
- new app walkthrough call
- new diagrams based on updated workflow
- handoff guide for Robocore staff
- additional courses

## Out of scope for this kind of project
- real LMS
- real logins/accounts
- admin editor
- database
- enrollments
- quizzes
- certificates
- reports

When a feature is out of current scope, explain the simpler alternative that still validates the guide.
