# Start Here

Use this folder as the planning/context pack for Claude Code.

## What to do
1. Create a new project folder called `robocore-guide-library`.
2. Copy these files into that folder.
3. Copy the Morpheus reference screenshots into `/reference`.
4. Open the folder in VS Code.
5. Open Claude Code in VS Code.
6. Paste the contents of `FIRST_CLAUDE_PROMPT.md`.
