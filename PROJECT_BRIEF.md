# Robocore Guide Library: Project Brief

## One-sentence goal
Build a polished web-based guide library for Robocore, starting with the Morpheus Drive course.

## What this is
A course-style web guide with a directory, a Morpheus Drive course page, lesson pages, reference content, and local progress behavior.

## What this is not
This is not a full LMS, admin dashboard, database app, or account system.

## Current paid scope
- Course directory/home page
- Morpheus Drive course overview
- Lesson 1 fully built from existing content
- Lesson 2 and Lesson 3 shown as planned/pending modules because the app workflow changed
- Reusable course/lesson layout so more courses can be added later
- Local progress using browser localStorage only
- Responsive desktop/tablet/mobile layout
- Deployment-ready static app

## Out of scope unless separately approved
- Real logins
- User accounts
- Database
- Admin editor
- Course creation UI
- Enrollments
- Quizzes
- Certificates
- Reports
- Writing full content for Lessons 2 and 3
- Creating new app diagrams/screenshots from the updated app
